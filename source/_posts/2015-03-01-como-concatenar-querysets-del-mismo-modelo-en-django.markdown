Modo #1: 

Podemos usar <code>itertools.chain</code> para  unir dos o más querysets:

```python
    from itertools import chain
    for item in chain(qs1, qs2, qs3):
        # ...
```

Modo #2:

Podemos usar el operador lógico ``OR``:

```python
    res = qs1 | qs2 | qs3
    res = res.distinct()
```
